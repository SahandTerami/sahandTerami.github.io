Problem Defenition:

This course project considers a quarter-car vibrational model with an input force and output displacement. The goal is to simulate the open-loop system and design various classical controllers (P, PI, PID) as well as an advanced state-vector feedback controller (SVFC). A full-order state observer is used in the advanced control system to estimate the system states.

  • Simulate the open-loop system response under initial conditions.
  • Design and implement classical controllers (P, PI, PID) to satisfy design requirements:
    • Settling time less than 5 seconds for a step input.
    • Maximum overshoot limited to 10%.
    • Zero steady-state error (or maximum 1% if present).
  • Design an advanced state-vector feedback controller (SVFC) if possible and implement it to meet the above requirements.
  • Select the most suitable controller and justify the choice.
  • Design a full-order continuous-time state observer to estimate system states and validate its performance.
