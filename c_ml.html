Module I: Fundamentals & Classical Learning

A deep dive into Supervised and Unsupervised learning, covering essential taxonomies from classification to clustering.

  • Concept & Instance-based Learning
  • Bayesian Theory & Decision Trees
  • Ensemble Methods (Bagging, Boosting, AdaBoost)
  • Bio-inspired Computing & ANN
  • Advanced Clustering (K-Means, Hierarchical)

Module II: Reinforcement Learning (RL)

Mastering the Agent-Environment interaction through the lens of modern optimization.

  • Core Logic: Exploration vs. Exploitation, Policy Gradient, and Reward Systems.
  • Mathematical Framework: Markov Decision Processes (MDP) & Bellman Equations.
  • Algorithms: Dynamic Programming, Monte Carlo Methods, and Temporal Difference (Q-Learning, SARSA).
  • Advanced Topics: Eligibility Traces & POMDPs.

🚀 Real-World Implementation Cases

  • Adaptive Email Routing using Markov Models.
  • Focused Web Crawlers driven by RL agents.
  • Learning-to-Rank algorithms for Search Engines.
  • Hands-on: RL implementation with MDP Toolbox (MATLAB).
