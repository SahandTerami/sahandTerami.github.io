Controlling the Vehicle’s Active Suspension System Using LQR and LQG (Master Studies)

  • Course Name: Modern Control II
  • May 2019 - June 2019

Problem Defenition:

This project focuses on designing an optimal control strategy for an active quarter-car suspension system to improve ride comfort, suspension travel, and road handling under road disturbances. Requirments of the project are as follows

  • Design and implement an LQR controller for an active quarter-car suspension system.
  • Simulate and analyze the passive and active suspension responses under a road bump disturbance.
  • Evaluate system performance in terms of ride comfort, suspension travel, and road handling.
  • Design and simulate a Kalman filter for state estimation.

Software and Frameworks:
