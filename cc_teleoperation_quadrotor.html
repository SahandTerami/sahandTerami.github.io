Design, Fabrication, and Implementation of a Pilot-in-the-Loop Teleoperation System for a Custom-Built Quadrotor

  • Course Name: Robotics and Lab. (Bachelor Studies)
  • November 2016 - February 2017

Problem Defenition:

This course project focuses on the design, build, and control of a quadrotor aerial vehicle operated by a human user via a joystick. The system must accurately translate the operator’s commands into stable and responsive flight. In this Project we should

  • Design and build a quadrotor aerial vehicle.
  • Implement and tune a control system for stable flight.
  • Integrate joystick-based human-in-the-loop control.
  • Test and validate the system during real-time operation.

Software and Frameworks:

  • C++
